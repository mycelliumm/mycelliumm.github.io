What is this Website?

This website is the final class project for a course in UMass Amherst's Professional Writing and Technical Communication certificate program. This website was made using the technical writing software MadCap Flare. The theme of the course, and of much of the program, is what 'accessibility' is and why it matters. The content of this website reflects the answers to those questions.

This website contains three main subjects: accessibility as a general term, a highlight of the Unity video tutorial project done earlier in the semester, and a deeper dive into audio description, a common but often misunderstood accessibility tool.

To navigate to the landing page for the Unity Video Project, click the following link: Unity Video Project Home.

To navigate to the landing page for the section of the website on audio description, click the following link: Audio Description Home.
